## 🚀 项目作用

本项目通过一段轻量级 JavaScript 脚本：

- 自动勾选必选项  
- 自动填写课程 & 教师评估文字内容  
- 智能匹配当前评估页面  
- 几秒钟完成整页评估

## 🧭 使用教程（超详细，小白友好）

### Step 1️⃣：进入课程评估页面

打开 **国科大选课系统**，进入你要填写的：  
- 「课程评估」或  
- 「教师评估」页面  

📸 示例截图：  
<img src="./assets/1.png" alt="进入评估页面示意图" />

---

### Step 2️⃣：打开浏览器控制台

在评估页面中：

- **Windows / Linux**：`F12` 或 `Ctrl + Shift + I`
- **macOS**：`Command + Option + I`

然后切换到 **Console（控制台）** 标签页

📸 示例截图：  
<img src="./assets/2.png" alt="打开浏览器控制台示意图" />
<img src="./assets/3.png" alt="打开浏览器控制台示意图" />

---

### Step 3️⃣：复制代码并粘贴运行

1. 打开本仓库中的脚本文件  
2. **全选复制代码**
3. 粘贴到浏览器控制台  
4. 按下 **Enter 回车键**

📸 示例截图：  
<img src="./assets/4.png" alt="控制台粘贴代码示意图" />

---

### Step 4️⃣：完成 🎉

此时你会看到：

- 所有必填项已自动勾选 ✅  
- 文本框已自动填写 ✍️  
- 页面无需刷新 📄

## 🧠 原理说明（可选阅读）

- 通过 DOM 查询定位表单元素  
- 自动触发点击 & 填充文本  
- 根据页面标题区分「课程评估 / 教师评估」  
- 不收集、不上传任何数据，**仅在本地运行** 🔒
